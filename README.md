# Taglish Sentiment and Emotion APIs

## Runs on http://127.0.0.1:8080/

## Endpoints: 
1. /sentiment
- returns a string (Positive, Negative or Neutral)

2. /emotion
- returns an array ([Love, Joy, Sadness, Fear, Anger, Surprise])
